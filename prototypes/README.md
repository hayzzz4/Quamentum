# Prototypes

## quamentum-prototype.html (2026-07-15)

Clickable UX prototype for the flows described in
`docs/superpowers/specs/2026-07-15-strava-training-insights-design.md`:
sign-in, the daily "read" with an AI suggestion (accept/edit/reject), and the
weekly plan manifest with a per-day workout builder covering all five sports.

Published (private) as a Claude Artifact for review:
https://claude.ai/code/artifact/9eca0987-590d-4129-aa28-fab166dfd0b3

Open the HTML file directly in a browser to view the same prototype locally —
it's self-contained, no build step or server required.

Status: awaiting UX feedback before moving to the implementation plan.
